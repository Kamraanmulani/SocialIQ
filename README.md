# SocialIQ a Social Media Analysis Chatbot

This project implements a **Social Media Analysis Chatbot** using **Streamlit** for the user interface, **Langflow** for AI-based analysis, and **Astra DB** for data management. The chatbot provides insights on social media content performance, including engagement metrics and other analysis based on the user's queries.

## Features

- **Chatbot Interface**: User-friendly interface built using Streamlit to interact with the bot.
- **Social Media Analysis**: Integration with Langflow API to analyze social media content based on user input.
- **Personalized Insights**: The chatbot provides personalized responses to social media content queries.
- **Data Management**: Uses Astra DB for managing social media data (comments, likes, shares, etc.).

## Installation

### Prerequisites

Before running the project, ensure you have the following installed:
- **Python 3.7+**
- **Streamlit**
- **Requests**
- **python-dotenv**
- **streamlit-chat**
